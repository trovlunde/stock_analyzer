## Codebase Patterns
- Use `uv run pytest` from repo root
- Cache keys use namespaced format: `{namespace}:{ticker}:{kind}`
- SqlCacheStore.get() returns None on miss; call .put() to store
- yfinance `.financials` / `.quarterly_financials`: index=line items, columns=period timestamps
- Monkeypatch `yfinance.Ticker` (the module attr) to block all network calls in tests
- `stock_analysis/ai/` has no __init__.py — namespace package, relative imports work in Python 3.12+
- edgartools `StitchedStatement.to_dataframe()` returns DataFrame with metadata cols (label/concept/standard_concept/preferred_sign) + date-string period cols; use `standard_concept` for concept mapping
- Monkeypatch `edgar.Company` and `edgar.MultiFinancials.extract` to mock all SEC HTTP in tests
- `EntityFilings.empty` is a property (len check); `EntityFilings.head(n)` returns first N filings
- CompositeProvider (fundamentals/) wraps YFinanceAdapter + EdgarAdapter; fallback fires on empty DataFrame
- Use `get_ticker_financials(str)`, `get_ticker_quarterly_financials(str)`, `get_quarterly_balance_sheet(str)` from helpers.py — do not call yfinance Ticker.financials directly
